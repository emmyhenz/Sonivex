# 🎵 Sonivex

## ✨ Features

✅ High-quality cloud audio streaming  
✅ Music downloads & offline playback  
✅ Favorites playlist system  
✅ Powerful search engine  
✅ Smooth animated premium UI  
✅ Mobile-first responsive design  
✅ Progressive Web App (PWA) support  
✅ Fast & lightweight

## Step 2: Deploy instantly

Click below to clone + deploy automatically:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/sonivex)

https://github.com/emmyhenz/sonivex

---

# 🛠 Local Installation

Clone repository:

```bash
git clone https://github.com/YOUR_USERNAME/sonivex.git
```

Enter folder:

```bash
cd sonivex
```

Run locally:

```bash
open index.html
```

or use live server.
